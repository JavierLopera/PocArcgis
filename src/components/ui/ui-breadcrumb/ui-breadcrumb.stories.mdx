## Story definition

Preview of default state.

<Story name="Develop">
  {() => {
		const knobs = {
			literals: object('literals', [
				{
					"href":"Ejemplo",
					"name":"Inicio"
				},
				{
					"href":"Ejemplo",
					"name":"Mis envios"
				},
				{
					"name":"Seguimiento"
				}
			])
		}
		return html`
			<correos-ui-breadcrumb literals='${JSON.stringify(knobs.literals)}'>
			</correos-ui-breadcrumb>
		`}
	}
</Story>
